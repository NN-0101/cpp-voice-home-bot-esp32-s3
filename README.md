# VoiceHomeBot

桌面智能语音机器人 —— 能聊、能看、能控家，你的桌面智能伙伴。

## 🎯 项目简介

VoiceHomeBot 是一款基于 ESP32-S3 的桌面级智能语音机器人。它集成了语音交互、表情显示、眼球运动和智能家居控制功能，既可以作为你的语音助手进行日常对话，也能通过语音命令控制家中的各类智能设备。一个摆在桌面上的机器人伙伴，让智能家居从“控制”变成“对话”。

## ✨ 核心功能

### 🗣️ 语音交互
- 语音输入：实时采集语音，支持离线/在线语音识别
- 语音输出：播放语音回复，声音自然流畅
- 对话能力：支持简单的问答与闲聊

### 🎭 生动表情
- OLED 屏幕显示动态表情（正常、微笑、闭眼、惊讶等）
- 周期性眨眼动画，让机器人更生动自然
- 可扩展表情库，配合语音状态变化

### 👀 灵动眼球
- 双舵机控制眼球水平/垂直转动
- 模拟真实目光追随，提升互动感
- 可与语音方向联动

### 🏠 家居控制
- 语音指令控制家电设备（灯光、空调、窗帘等）
- 支持自定义场景模式（离家、观影、睡眠等）
- 可接入主流智能家居平台

### 🌐 连接能力
- Wi-Fi 联网，支持配网与自动重连
- WebSocket 远程控制与状态查询
- 串口调试输出，便于开发调试

| 舵机 | SG90 / MG996R | 2 | 水平/垂直方向控制 |
